# 🛡️ davinci0x01_lvl0_Tasks

## 📝 About This Repository
This repository contains my hands-on solutions and scripts for the **Practical Cybersecurity Foundation - Level 0** training program by **The Hacker Association**. 

The primary objective of this repository is to document my journey from theoretical concepts to practical, code-based execution across various domains of cybersecurity, including scripting, system administration, network analysis, and applied cryptography.

> **Philosophy:** *"Go and get it - لا تنتظر كل شيء جاهزاً!"* > All tasks were solved through active research, documentation reading, and practical trial and error.

---

## 🚀 Modules & Skills Covered

### 🐍 Skill 1: Python
Practical Python scripting tailored for cybersecurity automation.
* **Level 1:** Variables, Data Types, Strings, and Input/Output operations.
* **Level 2:** Control Flow, For/While loops, and Logical Operators (e.g., bruteforcing PINs, building login systems).
* **Level 3:** Lists, Tuples, and Dictionaries (e.g., port/service mapping, HTTP status dictionaries).
* **Level 4:** Functions, Modules, and automation (e.g., MAC/Device ID generation, XOR encryption).

### 🌐 Skill 2: Networks
Understanding network fundamentals and building custom network scripts.
* **Day 1:** IP Addresses (IPv4/IPv6 validation), Subnetting, and CIDR Notation.
* **Day 2:** Ports & Protocols, building basic TCP/UDP port scanners.
* **Day 3:** Socket programming, connection-oriented vs. connectionless communication, and simulating basic network attacks/sniffing.

### 🪟 Skill 3: Windows OS Internals
System enumeration and administration on Windows environments using Python.
* **Day 1:** OS Detection, Basic System Info retrieval, and Environment Variables.
* **Day 2:** Process Listing, Management, and Services Control (e.g., building a task manager clone).
* **Day 3:** File System interaction, metadata extraction, and automation (e.g., directory monitoring, automated backups).

### 🐧 Skill 4: Linux OS (Bash)
Command-line mastery and system administration in Linux.
* **Day 1:** System Basics, hardware enumeration, and live resource monitoring.
* **Day 2:** Files & Permissions manipulation, ownership changes, and finding specific files.
* **Day 3:** Processes & Services management using `systemctl` and `journalctl`.

### 🔐 Skill 5: Cryptography
Practical implementation of secure cryptographic protocols.
* **Day 1:** Encoding (Base64/Hex) & Hashing Basics (MD5, SHA-1, SHA-256).
* **Day 2:** Symmetric Encryption using AES (ECB/CBC modes) and DES.
* **Day 3:** Asymmetric Cryptography (RSA Key Generation, Signing), and building a complete Hybrid Encryption system (RSA + AES).
